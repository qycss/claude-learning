## 1. Role Definition & Differentiation

**AI Agent Engineer** focuses on the Agent **application layer**: building Agent products, designing Prompt strategies, integrating RAG, optimizing Agent behavior, and evaluating Agent quality.
**AI Agent Infrastructure Engineer** focuses on the Agent **infrastructure layer**: Runtime engine, process isolation, security sandbox, retry engine, protocol stack.

The two roles overlap (e.g., Tool Use, Context Management) but differ in core focus.
